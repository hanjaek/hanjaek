### [DeepGPR-3D](https://github.com/hanjaek/DeepGPR-3D) (2025.09 ~ 2025.12)
> 싱크홀 분석 자동화 및 시민 대상 위험 지역 지도 제공 시스템

#### Tech Stack
![Python](https://img.shields.io/badge/Python-f5f5f5?style=flat)
![PyTorch](https://img.shields.io/badge/PyTorch-f5f5f5?style=flat)
![NumPy](https://img.shields.io/badge/NumPy-f5f5f5?style=flat)
![React](https://img.shields.io/badge/React-f5f5f5?style=flat)
![Vite](https://img.shields.io/badge/Vite-f5f5f5?style=flat)

#### Key Contributions
- GPR 데이터를 활용한 지반 공동(cavity) segmentation 모델 개발
- BCE + Dice Loss 적용을 통해 Dice +14%, IoU +11% 성능 개선
- SDT 기반 지반 공동 구조 3D 복원 구현
- React/Vite 기반 시민 대상 싱크홀 위험 구간 지도 프로토타입 구현

#### Achievements
- 분석 작업 자동화: **분석 시간 90% 이상 단축**
- 부산 지역문제 B.SORI 경진대회 **최우수상** 수상
